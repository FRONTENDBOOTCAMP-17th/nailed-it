# NAILED IT

Nail your frontend skills with quizzes.

## 📌 Overview

NAILED IT은 프론트엔드 학습자를 위한 퀴즈 기반 웹 서비스입니다.
프론트엔드 개념을 문제 풀이를 통해 학습할 수 있도록 하는 것을 목표로 합니다.

## 🚀 Goals

- 프론트엔드 개념을 퀴즈 형태로 학습
- 직관적인 UI/UX 제공
- 지속적인 기능 확장

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript

## 💻 Getting Started

프로젝트를 로컬 환경에서 실행하려면 아래 단계를 따라주세요.

```bash
# 1. 저장소 클론
git clone https://github.com/FRONTENDBOOTCAMP-17th/nailed-it-quiz.git

# 2. 폴더 이동
cd nailed-it-quiz

# 3. 의존성 설치 (npm 사용 시)
npm install

# 4. 프로젝트 실행
npm run dev
```

> This repository is created for a frontend quiz platform project.
